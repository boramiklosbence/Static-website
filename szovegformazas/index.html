Font-family:

A font-family tulajdonság segítségével beállítható a megjelenítendő szöveg betűtípusa.

Előfordulhat, hogy a felhasználó eszközére még nem lett telepítve a kívánt betűstílus. Ennek megoldása érdekében a font-family tulajdonság után több értéket is érdemes felsorolni, egy úgynevezett prioritás listát. A böngésző a listából az első olyan betűtípust fogja kiválasztani, amelyikkel meg tudja jeleníteni a szövegedet.

A megadható értékek típusai:

  • Betűtípuscsalád:
    • A betűtípusok megadására a fontok neve szolgál
    • A Microsoft és az Apple oprendszerek hasonló TrueType betűtípust használnak, a UNIX alapú rendszerek Type1 és PostScript fontokkal dolgoznak.
    • Ha mindenkinek kedvére akarunk tenni, akkor a webre kerülő output állományainkban TrueType betűtípusokat kell szerepeltetni, ugyanakkor biztosítani kell a lehetőséget azoknak is, akik nem ilyen fontokat használnak.
  • Általános betűtípuscsalád:
    • Egy másik lehetséges módszer, hogy olyan általánosan használt betűtípust állítunk be, melyet az összes rendszeren futó böngésző tud kezelni
    • Ilyen betűtípusból összesen 5 darab létezik (pl.: talpas karaktereket használó serif - Times New Roman)

HTML

<html>
<head>
 <title> Teszt </title>
 <link rel=”stylesheet” href=”./css/index.css”>
</head>
<body>
 <p>Alma</p>
 <p id=”szöveg”>Körte</p>
</body>
</html>

CSS

#szöveg {
 font-family: 'Times New Roman', Times, serif;
}

Font-size:

A font-size tulajdonság segítségével beállítható a megjelenítendő szöveg mérete.

  • Relatív érték megadás (ajánlott): az értékek a szülő elem méretéhez képest relatívan adják meg a betűméretet
    • %
    • em
    • smaller
    • larger
  • Abszolút érték megadás:
    • xx-small
    • x-small
    • small
    • medium
    • large
    • x-large
    • xx-large
    • px
    • pt

HTML

<html>
<head>
 <title> Teszt </title>
 <link rel=”stylesheet” href=”./css/index.css”>
</head>
<body>
 <p> Alma </p>
 <p id=”körte”> Körte </p>
 <p id =”barack”> Barack </p>
 <p id=”cseresznye”> Cseresznye </p>
</body>
</html>

CSS

#körte {
 font-size: smaller;
}

#barack {
 font-size: larger;
}

#cseresznye {
 font-size: 20px;
}

Font-weight:

A font-weight tulajdonság segítségével beállítható a megjelenítendő betűk vastagsága.

  • bold (félkövér)
  • bolder (félkövérnél vastagabb)

  • 100 - 900, a kerek 100-as értékek

Text-decoration:

A text-decoration tulajdonság segítségével klasszikus formázásokat hajthatunk végre a szövegen.

Megadható értékek:

  • none (nincs formázás)
  • underline (aláhúzás)
  • overline (fölé húzás)

  • blink (villogás - nem minden böngésző támogatja)
